Content 관련 메소드

* html(["문구"]) : innerHTML 속성과 비슷

선택된 요소의 content영역의 값을 리턴해주거나 또는 변경해주는 메소드
> $("선택자").html() : 선택된 요소의 content값을 반환해줌(html태그까지 반환)
> $("선택자").html("문구") : 선택된 요소의 content영역에 문구를 넣어줌(문구안에 html태그가 있으면 태그로 해석)

* text(["문구"]) 메소드 : innerText와 비슷

선택된 요소의 content영역의 값을 리턴해주거나 변경해주는 메소드
> $("선택자").text() : 선택된 요소의 content영역의 값을 반환(html태그가 있어도 그냥 문자로 인식)
> $("선택자").text("문구") : 선택된 요소의 content영역에 문구를 넣어(변경)줌(html태그가 있어도 그냥 문자로 넣어줌)
