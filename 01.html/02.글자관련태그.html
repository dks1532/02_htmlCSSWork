문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p 태그와 pre태그가 있다. p태그는 줄바꿈하고자 한다면 별도의 태그를 작성해야됨
그리고 공백은 한개의 공백만을 표시하기 때문에 별도의      기호를 기술해야 한다

HTML5는 HTML의 완전한 5번째 버전으로 월드 와이드 웹 (World Wide Web)의 핵심 마크업 언어이다. 2004년 7월 Web Hypertext Application Technology Working Group(WHATWG)에서 웹 애플리케이션 1.0이라는 이름으로 세부 명세 작업을 시작하였다. 

            pre 태그는 시작태그부터 종료태그까지 입력된 내용 그대로 출력해줌
            줄바꿈이라던가 혹은     여러개의 공백을 그대로 인식
            그래서 시를 적어보겠습니다
            
            [단심가]
            -정몽주-

            이 몸이 죽고 죽어 일백 번 고쳐 죽어

            백골이 진토되어 넋이라도 있고 없고

            님 향한 일편단심이야 가실 줄이 있으랴.

그 밖에 글자를 다루는 태그들

일반 글꼴 (태그로 감싸지 않는 텍스트)

b: 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더

i : 글자를 기울여서 표시하는 태그

em : 글자를 기울여서 표시하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그
sup : 윗첨자를 나타내는 태그

HTML란 월드 와이드 웹 (World Wide Web)의 핵심 마크업 언어이다.

글자 관련 태그 응용(태그를 중첩해서 사용 가능)

태그를 중첩으로 사용 가능하다.
굵게, 기울임 글꼴로 표현 가능하고,
